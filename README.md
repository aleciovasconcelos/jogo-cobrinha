# 🐍 Snake Professional

Uma versão moderna, responsiva e totalmente funcional do clássico jogo da Cobrinha (Snake), desenvolvida utilizando apenas **HTML, CSS e JavaScript Vanilla**, sem dependências externas.

---

## 📖 Sobre o Projeto

Este projeto recria um dos jogos mais icônicos da história dos videogames utilizando tecnologias web nativas.

O objetivo foi desenvolver uma versão visualmente agradável, organizada, performática e fácil de manter, mantendo a simplicidade de execução: basta abrir o arquivo HTML no navegador para jogar.

## 📈 Mecânica de Pontuação

* Cada comida consumida adiciona 1 ponto.
* A cobrinha aumenta de tamanho.
* A velocidade aumenta gradualmente.
* O jogo se torna mais desafiador conforme a pontuação cresce.

```text

```

---

Clone o repositório:

```bash

```

Entre na pasta:

```bash
cd snake-professional
```

Abra:

```text
snake.html
```
